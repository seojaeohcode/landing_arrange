주요 기능

✏️

스마트 제목 생성

긴 제목을 간결하고 검색하기 좋게 자동 변환합니다. 핵심 키워드는 유지한 채 읽기 쉽게 다듬어 줍니다.

📂

자동 카테고리 분류

폴더를 직접 만들 필요 없이, 관련 북마크끼리 알아서 묶어 줍니다. 정리가 한결 수월해집니다.

📈

사용 통계

자주 방문하는 북마크를 한눈에 확인할 수 있습니다. 방문 횟수와 카테고리별 통계를 제공합니다.

🔍

빠른 검색

북마크가 많아도 검색으로 곧바로 찾을 수 있습니다. 제목과 URL 모두 검색할 수 있습니다.

왜 A!rrange를 사용해야 할까요?

사용 방법

1

북마크 추가

크롬 툴바의 A! 아이콘을 클릭하고
사이드 패널에서 + 버튼을 눌러
현재 페이지를 저장합니다.

→
2

자동 처리

페이지 내용을 분석하여 제목을 최적화하고,
관련 북마크와 함께 적절한
카테고리로 분류합니다.

→
3

정리 완료

북마크가 자동으로 정리되어
카테고리별로 표시됩니다.
검색과 통계 기능을 바로 사용할 수 있습니다.
