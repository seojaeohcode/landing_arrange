스마트 제목 생성

긴 제목을 간결하고 검색하기 좋게 자동 변환합니다. 핵심 키워드는 유지한 채 읽기 쉽게 다듬어 줍니다.

자동 카테고리 분류

폴더를 직접 만들 필요 없이, 관련 북마크끼리 알아서 묶어 줍니다. 정리가 한결 수월해집니다.

사용 통계

🔍

빠른 검색

북마크가 많아도 검색으로 곧바로 찾을 수 있습니다. 제목과 URL 모두 검색할 수 있습니다.

사용 방법

패치노트

v1.1.0 업데이트 및 UI/UX 개선 내역입니다.

1. 신규 기능

  • 죽은 링크(Dead Link) 검사: 접속 불가능한 북마크 식별 및 선택 삭제 기능 추가
  • 북마크 데이터 백업: 로컬 파일 백업 및 내보내기 기능 구현
  • 폴더 관리 편의성 강화:
    • 폴더 삭제 시 내부 북마크 처리 방식(미분류 이동 또는 전체 삭제) 선택 기능 추가
    • 폴더 내 모든 북마크를 새 탭으로 한 번에 열기 기능 추가
  • 일괄 북마크 저장: 현재 열려 있는 모든 탭을 한꺼번에 북마크로 저장하는 기능 구현
  • 공지사항 시스템: 상단 알림 아이콘을 통한 업데이트 및 새 소식 확인 기능 도입

2. UI/UX 개선

  • 디자인 리뉴얼: 설정 페이지 및 대시보드 레이아웃 직관적 재구성
  • 커스텀 모달 적용: 브라우저 기본 알림창을 Arrange 전용 모달로 교체하여 시각적 일관성 확보
  • 용어 체계 정립: 기존 '카테고리' 명칭을 '폴더'로 일괄 변경
  • 레이아웃 최적화: 반응형 스타일 조정 및 불필요한 스크롤 제거

3. 주요 버그 수정

  • 검색 결과에서 관련 없는 빈 폴더가 노출되는 현상 수정
  • 특정 상황에서 북마크가 '미분류'로 오분류되는 현상 해결
  • 북마크 재가져오기 시 기존 데이터 초기화 및 중복 색상 표시 오류 수정
  • 모달 내 텍스트 줄바꿈 및 알림창 잘림 현상 수정
  • 안정성 및 사용성 개선
  • 스마트 제목 요약
  • 자동 카테고리 분류
  • 드래그 앤 드롭 정리
  • 다크 모드, 사용 통계, 빠른 검색
