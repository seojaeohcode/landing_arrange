주요 기능

✏️

스마트 제목 생성

긴 제목을 간결하고 검색하기 좋게 자동 변환합니다. 핵심 키워드는 유지한 채 읽기 쉽게 다듬어 줍니다.

빠른 검색

북마크가 많아도 검색으로 곧바로 찾을 수 있습니다. 제목과 URL 모두 검색할 수 있습니다.

사용 방법

패치노트

이번 업데이트에서 추가·개선된 내용이에요.

✨ 새로 생긴 것들

  • 피드백·후원 버튼 — 오른쪽 아래에 ✉️(의견/버그 제보), ☕️(후원) 버튼이 생겼어요. 불편한 점이나 아이디어를 편하게 보내주세요.
  • 한 번에 더 많이 정리 — 아직 분류 안 된 북마크를 이전 100개에서 500개까지 한 번에 AI가 정리해줘요.
  • 중복 북마크 방지 — 같은 주소의 북마크가 두 번 쌓이지 않도록 막아요. 중복일 때 안내 문구도 띄워줘요.

🔧 고쳐진 것들

  • 북마크를 클릭해도 방문 횟수가 안 올라가던 문제를 수정했어요.
  • 북마크가 많을 때 오류나 멈춤이 나오던 현상을 줄였어요.
  • 죽은 링크 검사 등에서 창이 갑자기 닫히던 현상을 고쳤어요.

💡 더 편해진 점

  • 여러 창이 겹쳐 보이던 걸 정리해서 화면이 더 깔끔해졌어요.
  • 죽은 링크 검사 버튼을 다른 버튼들과 스타일을 맞춰 통일감을 줬어요.
  • 피드백·후원 버튼을 누르기 쉽고 보기 좋게 다듬었어요.

이번 업데이트에서 추가·개선된 내용이에요.

✨ 새로 생긴 것들

  • 죽은 링크 검사 — 더 이상 열리지 않는 북마크를 찾아서 골라 삭제할 수 있어요.
  • 북마크 백업 — 북마크 목록을 파일로 저장해 두었다가 나중에 다시 불러올 수 있어요.
  • 폴더 관리가 더 쉬워졌어요
    • 폴더를 지울 때, 안에 있는 북마크를 «미분류로 보내기» vs «함께 삭제» 중에서 고를 수 있어요.
    • 폴더 안 북마크를 한 번에 새 탭으로 열 수 있어요.
  • 탭 한 번에 저장 — 지금 열려 있는 탭을 한 번에 전부 북마크로 저장할 수 있어요.
  • 공지 알림 — 상단 알림으로 업데이트 소식과 새 소식을 확인할 수 있어요.

💡 더 편해진 점

  • 설정 화면과 대시보드를 더 알아보기 쉽게 바꿨어요.
  • 알림 창을 A!rrange 스타일에 맞게 통일했어요.
  • «카테고리» 대신 «폴더»로 통일해서 보여줘요.
  • 화면 크기에 맞게 잘 보이도록, 불필요한 스크롤이 덜 나오도록 손봤어요.

🔧 고쳐진 것들

  • 검색했을 때 빈 폴더가 섞여 나오던 걸 고쳤어요.
  • 어떤 북마크가 잘못 «미분류»로 들어가던 문제를 수정했어요.
  • 북마크를 다시 불러올 때 데이터가 사라지거나 색이 겹쳐 보이던 걸 고쳤어요.
  • 알림 창 안 글자가 잘리거나 줄바꿈이 이상하던 걸 수정했어요.
  • 오래 켜 두면 느려지던 현상을 줄였어요.
  • 정리 후 새로 생긴 폴더가 화면에 바로 반영되지 않던 문제를 고쳤어요.
  • AI 정리 중간에 오류가 나던 현상을 수정했어요.
  • 더 안정적으로, 쓰기 편하게 손봤어요.
  • 긴 제목을 핵심만 남겨 스마트하게 요약
  • 북마크를 자동으로 폴더에 분류
  • 드래그해서 순서·폴더 마음대로 정리
  • 다크 모드, 방문 통계, 빠른 검색
